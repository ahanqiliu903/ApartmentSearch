# Apartment Search Tool

This application aims to make apartment hunting in NYC easier.

Goals:\
-Pull listings from multiple sources (Zillow API, Rentcast, StreetEasy, etc.) while streamlining all preference data\
-Score/rank listings using a ML model\
-Email daily updates of new listings that match a certain score threshold (price, location, amenities, subway distance)\
-Log all listings to a Google Sheet\
-Flag suspicious/scam listings using agentic web searches and API lookups\
